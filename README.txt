GAITANV3 MLAPP file was developed with Matlab R2020a in Windows 11, version 22H2.
The MLAPP is annexed into the repository, as well as the executable, the executable can be only run in a Windows environment.
The MyAppInstaller_web.exe file to install the MATLAB Runtime from internet.
A subject's files are annexed as an example to test the app
An user´s manual for the GAITANV3 app is annexed, pleasee check it in order to use properly the app 

This code was developed to be used mainly in gait analysis and gait related research, it detects the gait events using only the signals from IMU's and compare them to VICON events to validate them.
